## 〇、產出資訊（讓 SELA 驗證）

- **使用 Kit 版本：** V1.21.0
- **使用範本：** §14 範本 C 教育型
- **資訊來源：** 本專案 `CLAUDE.md`〇/一/二（定位、技術棧、業務對映）+ `DESIGN.md`（北歐極簡設計）

## 一、萃取的設計 context

- **這個 app 做什麼：** 國中數學・物理・化學的複習網站，把每章拆成小課 + 互動測驗 + 圖解
- **使用情境：** 學生在手機/電腦上分科分章複習，邊看重點邊做題對答案
- **核心精神：** 北歐極簡、暖而不冷、留白、克制的層次（不花俏）
- **氣質方向：** 安靜、專注、學習感（非童趣卡通）
- **品牌歸屬：** SELA 個人專案（app logo 為主視覺、SELA logo 退居 README footer）

## 二、自動決策

### 範本類型
教育型（§14 範本 C）。但本站走北歐極簡、非童趣，logo 精神偏「沉靜的學習」而非鮮豔卡通。

### 壁虎/蜥蜴傾向
非必要。教育複習站壁虎不是自然契合的符號，是否帶入由生圖 AI 自行判斷（a gecko motif is welcome only if it fits, not required）。

### 背景色提案（圓角填色、坐落白底上）
`#2B4C7E`（深海板岩藍）— 本站數學科 accent，學術沉靜、與 SELA 橘明顯不同色（不撞色）。

## 三、完整 Prompt（複製這段貼到生圖 AI）

```
一個 app icon 的 logo。

這個 app 是什麼：
一個給國中學生用的數學、物理、化學複習網站。
把每一章拆成一小課一小課，附重點整理、互動測驗和圖解，
讓學生分科分章複習、邊看邊練習。

設計要求：
- 圓角方形、1:1 比例
- 扁平向量風格、單一白色主體圖案
- 底色填 #2B4C7E 深藍、放在純白背景上
- 不要漸層、陰影、3D、光暈
- 小尺寸（favicon）也要清楚

主體畫什麼由你發揮、呼應這個 app 的學習精神即可。
```

## 四、給 SELA 的備註

- 拿到圖先看背景：若是純白底（Gemini 通常給白底）→ 直接給我，我走 §10.2 floodfill 轉透明 + 縮各尺寸。
- 不滿意可換底色：站內現成色票有 數學藍 `#2B4C7E`、化學棕 `#7A4E36`、物理綠 `#3A5C42`、暖米 `#F4F2ED`。
- 套進來後：`favicon/` 整組換成 app logo，SELA logo 移到 README footer 當品牌歸屬（雙軌系統）。
